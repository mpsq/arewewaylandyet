# How to build

## Prerequisites

- [Hugo](https://github.com/gohugoio/hugo) 0.77+

## Steps

1.

```shell
git clone git@github.com:mpsq/arewewaylandyet.git
cd arewewaylandyet
```

2.

```shell
hugo server -D
```

3. The development server should be accessible at http://localhost:1313.
